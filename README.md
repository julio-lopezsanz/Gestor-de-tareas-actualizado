# Gestor de Tareas en Python

Un programa de consola simple para gestionar tareas diarias.  
Permite agregar, completar, eliminar y visualizar tareas de forma clara y eficiente.

Este proyecto fue creado como ejercicio práctico para reforzar fundamentos de Python como:
- listas
- funciones
- validación de entradas
- control de flujo

---

## Funcionalidades

- Agregar tareas
- Marcar tareas como completadas
- Eliminar tareas (completadas o no)
- Mostrar la lista actual de tareas

Las tareas completadas se muestran con un **check** al final, sin afectar la posibilidad de eliminarlas.

---

## Reglas del programa

- Las tareas se almacenan en una lista.
- No se utilizan librerías externas.
- Toda la lógica está separada en funciones.
- El programa se ejecuta en consola.

---
## Objetivo del proyecto

Este proyecto tiene fines educativos y forma parte del proceso de aprendizaje de Python, enfocándose en escribir código claro, funcional y fácil de mantener.
